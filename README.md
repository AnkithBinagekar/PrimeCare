### Option 2: Full Backend Setup

```bash

````

2. Move project folder into XAMPP `htdocs`

3. Create database

```sql
CREATE DATABASE primecare_db;
```

4. Import SQL file into `primecare_db`

5. Update database credentials in PHP configuration file

6. Start Apache and MySQL from XAMPP control panel

7. Open in browser:

```
http://localhost/PrimeCare
